# pi_finder_cpp
Pi Finder C++  Ein persönliches Projekt zur Demonstration von Hochpräzisionsberechnungen mit MPFR/GMP und Suchalgorithmen (KMP). Es berechnet Millionen von π-Dezimalstellen und findet beliebige Ziffernfolgen oder Geburtsdaten darin – mit mathematischer Erklärung und interaktiver Konsolenausgabe.
